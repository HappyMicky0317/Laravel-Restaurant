Installation

You will need to make sure your server meets the following requirements

  • PHP >= 5.5.9
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension

Linux Users

  • Unzip the main zip file and upload your public_html folder (Linux hosting user will not upload vendor folder)
  • Give 777 permission to storage/ and bootstrap and public/assets/images folder
    i.e (Linux: chmod -R 777 foldername)
  • Create database in phpmyadmin and import .sql file from DB folder
  • composer install
  • If you are using linux do not upload "vendor" folder

Shared Hosting Users

  • Unzip the main zip file and upload your public_html folder (Linux hosting user will not upload vendor folder)
  • Move Public folder index.php, .htaccess and assets folder to public_html
  • Give 777 permission to storage/ and bootstrap/ and public/assets/images folder
  • Create database in phpmyadmin and import .sql file from DB folder
  • On shared hosting upload "vendor" folder also
